# Energy Hive - Portable Solar-Powered Energy & Cooling Hub

## 🌍 Project Overview

Energy Hive is a revolutionary portable solar-powered energy and cooling hub designed for farmers, vendors, students, healthcare workers, and rural communities. It generates clean electricity through foldable solar panels, stores energy in rechargeable batteries, and provides thermoelectric cooling to preserve perishable goods while maintaining complete sustainability.

### 🎯 Core Features

- **⚡ Solar Power Generation**: Foldable solar panels for clean, renewable electricity
- **🔋 Smart Battery Storage**: Advanced battery systems for 24/7 power supply
- **❄️ Thermoelectric Cooling**: Preserve vegetables, milk, medicines, and perishables
- **📱 Device Charging**: Multiple USB ports and wireless charging capabilities
- **💡 LED Lighting**: High-efficiency LED lights for homes, farms, and clinics
- **🌱 Sustainable Energy**: 100% renewable, zero-emission technology

## 🔧 Customization Guide

### Update Team Information
Edit the Team Section in `index.html`:
```html
<p class="team-name">[Your Name]</p>
<p class="team-name">[Your School Name]</p>
<p>[Your City], [Your Country]</p>
<p>+91 XXXX-XXXX-XXXX</p>
<p>contact@energyhive.com</p>
```

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-green: #22C55E;
    --solar-yellow: #FACC15;
    /* Update other colors as needed */
}
```

### Add Images to Gallery
Replace placeholders in Gallery Section with actual images:
```html
<img src="path/to/image.jpg" alt="Description">
```

### Update Contact Form
Configure form submission in `script.js` to send emails via your backend service.
